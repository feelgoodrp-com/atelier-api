# Security policy

This service handles auth (Discord OAuth, device tokens, sessions) and storage —
please report security issues responsibly.

**Don't open a public issue.** Instead, report it privately through GitHub
[security advisories](https://github.com/feelgoodrp-com/atelier-api/security/advisories/new),
or contact a maintainer directly. We'll respond as soon as we can and credit you
if you'd like.
